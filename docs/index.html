About Banana Code

Banana Code is an innovative programming language designed to simplify the development process. It combines the efficiency of compiled languages with the simplicity of scripting languages, making it an ideal choice for both beginners and experienced developers.

Developed by a passionate team of programmers, Banana Code aims to address common challenges faced in software development, providing a robust platform for building versatile applications.

Fibonacci

Banana 
pick fibonacci(Int n) -> Int:
    if (n <= 1):
        serve n
    else:
        serve fibonacci(n-1) + fibonacci(n-2)

Num Is Even

Banana 
pick is_even(Int n) -> Boo:
    serve n % 2 == 0

String to Array

Banana 
let String sentence = 'This is a sentence.'
let Bunch words = sentence.peel(' ')

plant(words) // output: 0x00007F543210ABCD
plant(words!) // output: ['This', 'is', 'a', 'sentence.']

Person Class

Banana 
model Person:
    config(String name, Int age):
        self.name = name 
        self.age = age

    pick greet() -> Nothing:
        plant('Hello, my name is \!')

let Person person1 = Person('Alice', 30)
person1.greet() // Output: Hello, my name is Alice!

Positive or Negative with Ternary

Banana 
pick check-sign(Int num) -> String:
    let Boo result = (num > 0)? -> 'Positive' ->> 'Negative'
    serve result
